Western Ranger
	How to Play:
		Press Enter Key at Menu to start game
		Up-Down-Left-Right Arrows Navigate Menu
		Dusty's Movement - WSAD - W = Up, S = Down, A = Left, D = Right (Can be used in combinations to move Diagonal)
		Dusty's Running - Hold Shift and Use Movement Arrows
		Aim Dusty's Weapons - Use the Mouse Cruisor to Control Dusty's Weapon Aiming
		Shoot Dusty's Weapons - Left Click the Mouse
		Change Dusty's Weapons - 1 Key is Shotgun, 2 Key is Revolver, 3 Key is Rifle
		Dusty Eats Food - Q Key
		Dusty Interacts with NPC's - E Key
		
	Cheat Codes: 
		Space + M = Dusty Gains $99
		Space + N = Dusty Gains 99 of all Ammo Types
		Space + B = Warps Dusty to Boss Room
		Space + F = Dusty gains 99 Food
